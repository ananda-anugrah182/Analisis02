Menampilkan Gambar

Fungsi paket R dasar (“EBImage”) mencakup membaca, menulis, dan menampilkan gambar. Gunakan fungsi readImage untuk membaca gambar, yang menggunakan nama file atau URL sebagai input.Fungsi paket R dasar (“EBImage”) mencakup membaca, menulis, dan menampilkan gambar. Gunakan fungsi readImage untuk membaca gambar, yang menggunakan nama file atau URL sebagai input.

Package R (“EBImage”) saat ini mendukung tiga format file gambar: jpeg, png dan tiff. Format gambar tambahan dapat diimpor menggunakan R GitHubPkg (“aoles / RBioFormats”), yang menambahkan dukungan untuk berbagai format file termasuk data gambar mikroskopis dan metadata.

Berikut adalah cara menampilkan gambar menggunakan fungsi display()

Image <- readImage('picture.jpg')
display(Image)

Kita juga bisa menggunakan method = “browser” untuk menampilkan gambar, melalui fungsi ini, kita bisa memperbesar dan memperkecil saat menampilkan gambar

Dan berikut adalah cara menampilkan gambar menggunakan method=“browser”

img = readImage("picture.jpg")
display(img, method="browser")

Memanipulasi gambar

Operator aritmatika R dapat dengan mudah memanipulasi gambar, misalnya, kita dapat menghasilkan gambar negatif hanya dengan mengurangi gambar dari nilai maksimumnya.

img_neg = max(img) - img
display( img_neg )

Kita juga dapat meningkatkan kecerahan gambar melalui penambahan, menyesuaikan kontras melalui perkalian, dan menerapkan koreksi gamma melalui eksponen.

img_comb = combine(
  img,
  img + 0.3,
  img * 2,
  img ^ 0.5)
display(img_comb, all=TRUE)

Untuk transposisi gambar,dapat menggunakan fungsi transpose dalam pake R’s R (base") dengan fungsi t.hal ini juga dapat berfungsi pada gambar berwarna dan multiframe dengan cara menukar dimensi spasialnya

img_t = transpose(img)
display( img_t )

Transformasi Spasial

Kita hanya melihat satu jenis transformasi spasial, transpos, tetapi ada lebih banyak jenis, seperti terjemahan, rotasi, refleksi, dan penskalaan. Terjemahan menggunakan vektor dua dimensi yang ditentukan untuk memindahkan bidang gambar untuk memotong piksel yang berakhir di luar area gambar, dan menyetel pixel yang masuk ke area gambar sebagai latar belakang.

Translasi Gambar

img_translate = translate(img, c(100,-50))
display(img_translate)

Rotasi Gambar Penggunaan rotasi berfungsi untuk memutar gambar searah jarum jam sesuai dengan sudut yang diberikan

img_rotate = rotate(img, 30, bg.col = "white")
display(img_rotate)

Mengganti Ukuran Gambar Untuk menskalakan gambar ke dimensi sesuai keinginan, kita dapat mengubahan ukuran. Jika hanya mengubah salah satu dari lebar atau tinggi,pada dimensi lainya secara otomatis dihitung dengan tetap menggunakan aspek rasio asli.

img_resize = resize(img, w=256, h=256)
display(img_resize )

Refleksi Gambar Fungsi flip dan flop masing-masing mencerminkan gambar di sekitar sumbu horizontal dan vertikal gambar.

img_flip = flip(img)
img_flop = flop(img)

display(combine(img_flip, img_flop), all=TRUE)

Transformasi Gambar Transformasi linier spasial diimplementasikan menggunakan transformasi affine umum. Ini memetakan koordinat piksel gambar px menggunakan matriks transformasi 3x2 m. Misalnya, pemetaan tipis horizontal dapat diterapkan dengan:

m =  matrix(c(1, -.5, 128, 0, 1, 0), nrow=3, ncol=2)
img_affine = affine(img, m)
display( img_affine )

Filtering

Gambar biner adalah gambar yang hanya berisi dua set piksel, dengan nilai, katakanlah 0 dan 1, mewakili piksel latar belakang dan latar depan.

shapes = readImage(system.file('images', 'shapes.png', package='EBImage'))

Segmentasi gambar melakukan partisi gambar, dan biasanya digunakan untuk mengidentifikasi objek dalam gambar. Objek terhubung yang tidak bersentuhan dapat disegmentasi menggunakan fungsi tersebut bwlabel, sedangkan watershed dan propagate menggunakan algoritma yang lebih canggih dapat memisahkan objek yang saling bersentuhan.

Manipulasi Objek

Package R (“EBImage”) mendefinisikan objek sebagai sekumpulan piksel dengan nilai integer unik yang sama. Biasanya citra yang mengandung objek merupakan hasil dari fungsi segmentasi. Objek dapat dihapus dari gambar tersebut. Secara default, setelah penghapusan objek, semua objek yang tersisa diberi label ulang sehingga ID objek tertinggi sesuai.

objects = list(
    seq.int(from = 2, to = max(logo_label), by = 2),
    seq.int(from = 1, to = max(logo_label), by = 2)
    )
logos = combine(logo_label, logo_label)
z = rmObjects(logos, objects, reenumerate=FALSE)
display(z, all=TRUE)

Segmentasi Sel

Kesimpulan dari vignette ini yaitu dengan menerapkan fungsi yang dijelaskan sebelumnya ke tugas segmentasi sel. Tujuannya adalah untuk mengidentifikasi dan mengkarakterisasi secara kualitatif sel-sel dalam sampel citra mikroskop melalui penghitungan. Cara ini bisa diterapkan pada koleksi yang berisi ribuan gambar. Kita bisa memuat gambar inti dan badan sel. Untuk memvisualisasikan sel, kita melapisi gambar-gambar ini sebagai saluran hijau dan biru dari gambar berwarna palsu.

nuc = readImage(system.file('images', 'nuclei.tif', package='EBImage'))
cel = readImage(system.file('images', 'cells.tif', package='EBImage'))

cells = rgbImage(green=1.5*cel, blue=nuc)
display(cells, all = TRUE)

Pertama, yaitu segmentasi menggunakan thresh, fillHull, bwlabel dan opening.

nmask = thresh(nuc, w=10, h=10, offset=0.05)
nmask = opening(nmask, makeBrush(5, shape='disc'))
nmask = fillHull(nmask)
nmask = bwlabel(nmask)

display(nmask, all=TRUE)

Kemudian, menggunakan inti tersegmentasi sebagai benih dalam segmentasi Voronoi dari sitoplasma.

ctmask = opening(cel>0.1, makeBrush(5, shape='disc'))
cmask = propagate(cel, seeds=nmask, mask=ctmask)

display(ctmask, all=TRUE)

Untuk memvisualisasikan segmentasi dapat menggunakan paintObject.

segmented = paintObjects(cmask, cells, col='#ff00ff')
segmented = paintObjects(nmask, segmented, col='#ffff00')

display(segmented, all=TRUE)

Seni, Pemrosesan Gambar dan Representasi Spektral Sinyal

Kita coba menginterogasi gambar dibawah ini

library(EBImage)
image <- readImage('image.jpg')
display(image, method = "raster", all = TRUE)

# Intensity Histogram
hist(image)

# Generate negative image
img_neg <- max(image) - image
display( img_neg )

# dim: 633 1000    3

# crop and threshold
img_crop <- image  
img_thresh <- img_crop > 0.55
display(img_thresh)

# Low-pass Filter/smooth
w <- makeBrush(size = 31, shape = 'gaussian', sigma = 5)
img_flo <- filter2(image, w)
display(img_flo)

# High-pass filtering Laplace Filter/Edge detection
fhi <- matrix(1, nrow = 3, ncol = 3)
fhi[2, 2] <- -7
img_fhi <- filter2(image, fhi)
display(img_fhi)

# Denoising
img_median <- medianFilter(image, size=5, cacheSize=20000)
display(img_median)

# Segmentation
grayimage <- channel(image,"gray")
threshold <- otsu(grayimage)
CC_th <- combine( mapply(function(frame, th) frame > th, 
                         getFrames(image), 
                         threshold, SIMPLIFY=FALSE) )
display(CC_th, all=TRUE)

disc <- makeBrush(51, "disc")
disc <- disc/sum(disc)
offset <- 0.0001
CC_bg <- filter2(image, disc)
CC_th <- image < CC_bg + offset
display(CC_th, all=TRUE)

# Watershed segmentation
nmask <- watershed(distmap(CC_th), tolerance=5, ext=20)
display(colorLabels(nmask), all=TRUE)

# Voronoi image tesselation
voronoiExamp <- propagate(x = nmask, seeds = nmask, lambda = 0.001)
voronoiPaint <- colorLabels (voronoiExamp)
display(voronoiPaint, all=TRUE)

CC <- rgbImage(green=2.0*CC_bg, blue=img_median, red=voronoiPaint[ , , , 1])
display(CC, all = TRUE)

ctmask = opening(img_fhi<0.6, makeBrush(5, shape='disc'))
cmask = propagate(img_fhi, seeds=nmask, mask=ctmask)
display(ctmask, all=TRUE)

ctmask = opening(img_fhi>0.1, makeBrush(5, shape='disc'))
cmask = propagate(img_fhi, seeds=nmask, mask=ctmask)
display(ctmask, all=TRUE)

segmented = paintObjects(channel(cmask,"gray"), image, col='#ff00ff')
segmented = paintObjects(channel(ctmask,"gray"), segmented, col='#ffff00')
display(segmented, all=TRUE)
